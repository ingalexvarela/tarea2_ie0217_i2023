Detailed Description

template<typename T>
class Stack< T >

Una clase de plantilla que representa una pila de elementos genéricos.

Esta clase de plantilla representa una pila que puede contener cualquier tipo de objeto. Los elementos se agregan a la pila en la parte superior y se eliminan de la pila desde la parte superior.
